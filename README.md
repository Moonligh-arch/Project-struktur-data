# 📚 Simulasi Lift dengan Priority Queue

Proyek ini merupakan simulasi sistem lift (elevator) menggunakan **Priority Queue** yang diimplementasikan dalam Python. Fokus utama adalah mengaplikasikan algoritma **SCAN (Elevator Algorithm)** untuk menjemput penumpang secara efisien, sekaligus memperkenalkan konsep struktur data dinamis seperti nested dictionary.

## 🎯 Studi Kasus
- **Tujuan**: Menunjukkan bagaimana priority queue dapat mengoptimalkan penjadwalan lift pada gedung bertingkat.
- **Fitur Utama**:
  - Input interaktif untuk menambah penumpang (lantai asal & tujuan).
  - Visualisasi langkah demi langkah pergerakan lift.
  - Monitoring status lift, kapasitas, dan antrean penumpang.
  - Stress test untuk menguji performa pada beban tinggi.

## 🚀 Cara Menjalankan
```bash
python3 project_V4.py
```
Pastikan Python 3 terinstal. Ikuti menu yang muncul di terminal untuk mengoperasikan simulasi.
